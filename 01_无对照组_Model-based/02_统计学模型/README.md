# 统计学模型

## 适用条件

- 无对照组
- **不需要深度业务洞察**：只需估计处理效应大小，无需刻画经济机制

## 方法简介

利用时间序列或贝叶斯方法，以处理前数据（及相关协变量）构建反事实预测，再与处理后观测值对比。

### 典型方法

| 方法 | 说明 |
|------|------|
| BSTS (Causal Impact) | Google 开发，基于贝叶斯结构时间序列，适合单一时间序列的政策评估 |
| Prophet + 合成控制 | 用预测模型替代合成控制组 |
| Interrupted Time Series (ITS) | 分段回归，适合群体级干预 |

## 使用流程

1. 确认有足够长的处理前时间序列
2. 用处理前数据拟合 BSTS 模型（可引入控制变量/协变量）
3. 处理后：预测值 = 反事实，实际值 - 预测值 = 处理效应
4. 进行稳健性检验（安慰剂检验、敏感性分析）
