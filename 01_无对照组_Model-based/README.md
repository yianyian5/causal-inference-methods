# 无对照组 — Model-based 方法

## 适用条件

**无法找到对照组**时，退而求其次使用模型驱动的方法，依靠模型本身提供反事实。

## 决策分支

```
无对照组
  ├── 需要更深的业务洞察？
  │     ├── Y → 经济学模型 / 结构估计（见子目录 01）
  │     └── N → 统计学模型（见子目录 02）
```

## 子目录说明

| 目录 | 方法 | 关键特征 |
|------|------|----------|
| `01_经济学模型_结构估计` | BLP、供需结构模型等 | 基于经济理论，可解释，适合策略模拟 |
| `02_统计学模型` | BSTS / Causal Impact 等 | 数据驱动，适合时间序列干预评估 |
