# 经典 DID 设计

## 适用条件

- 存在不可观测混淆因素
- 干预为**一次性**（非交错），实验组与对照组分组清晰
- 有处理前和处理后至少各一期数据

## 决策分支

```
经典 DID
  ├── TWFE 估计（双向固定效应）
  │     ├── Event Study 事前趋势检查
  │     │     ├── 通过 → 结果可信，进入稳健性检验
  │     │     └── 未通过
  │     │           ├── 可通过控制变量 / 函数变换 / PSM 调整 → 重新尝试 TWFE
  │     │           └── 无法调整 → 转向 SCM / SDID
  │     
  └── SCM / SDID（合成控制 / 合成 DID）
```

## 子目录说明

| 目录 | 内容 |
|------|------|
| `TWFE/通过事前趋势检查` | TWFE 估计结果可信的标准流程 |
| `TWFE/未通过事前趋势_调整后重试` | 通过协变量控制等方式修复平行趋势后重新估计 |
| `TWFE/无法调平行趋势_转SCM_SDID` | 平行趋势无法满足，转向合成控制方法 |
| `SCM_SDID` | 合成控制法（SCM）与合成双重差分（SDID）的实现 |
