# 无法调整平行趋势 → 转向 SCM / SDID

## 状态

经过多种调整尝试，TWFE 的平行趋势假设**仍然无法满足**。  
此时应转向 `SCM_SDID/` 目录中的合成控制方法。

## 原因分析

平行趋势无法满足的常见原因：
- 处理组与对照组在处理前就已有系统性差异（如不同城市、不同渠道）
- 样本量较小，对照组无法充分代表处理组的反事实趋势
- 存在时变混淆因素（time-varying confounders）且无法观测或控制

## 替代方案

转向 `../SCM_SDID/`，使用：
- **SCM（合成控制法）**：通过加权组合多个对照单元构造"合成对照"
- **SDID（合成双重差分）**：结合 DID 和 SCM 的优点，对时间和单元同时加权

这些方法不要求严格的平行趋势，而是通过优化找到最佳对照，使得处理前拟合最优。
