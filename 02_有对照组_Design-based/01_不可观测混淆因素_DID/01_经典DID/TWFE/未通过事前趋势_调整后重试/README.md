# 未通过事前趋势检查 → 调整后重试 TWFE

## 状态

TWFE 的 Event Study 检查**未通过**，但可以通过以下方式修复平行趋势，重新进行 TWFE 估计。

## 常用调整方式

### 1. 加入控制变量
在 TWFE 回归中加入随时间变化的控制变量 $X_{it}$，从 **条件平行趋势** 的角度识别因果效应：
$$Y_{it} = \alpha_i + \lambda_t + \beta \cdot D_{it} + \gamma X_{it} + \varepsilon_{it}$$

### 2. 函数形式变换
- 对结果变量取对数（log transformation）
- 使用标准化或归一化后的结果变量
- 有时可消除量级差异带来的趋势分歧

### 3. PSM 预处理（Pre-matching）
在进行 TWFE 之前，用倾向得分匹配（PSM）构造更可比的对照组：
- 匹配后再做 TWFE，可大幅改善平行趋势
- 原理：通过限制样本提高处理组与对照组的协变量平衡

## 调整后验证

调整后，**必须重新进行 Event Study 检查**：
- 若通过 → 返回 `TWFE/通过事前趋势检查/`
- 若仍未通过 → 转向 `TWFE/无法调平行趋势_转SCM_SDID/`

## 注意事项

- 调整方式需有理论依据，不可"数据挖掘式"地搜索能通过检查的规格
- 每次调整均需报告，确保透明度
