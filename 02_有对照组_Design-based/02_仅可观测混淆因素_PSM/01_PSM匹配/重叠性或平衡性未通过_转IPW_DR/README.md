# 重叠性或平衡性未通过 → 转向 IPW / DR

## 状态

PSM 匹配存在以下问题之一：
- ❌ **样本损失不可接受**：匹配过程丢弃了过多样本，导致结论代表性不足
- ❌ **平衡性检查未通过**：匹配后处理组与对照组协变量仍存在系统性差异（SMD 超标）

## 后续方向

转向 `../../02_IPW_DR/` 使用加权方法，这些方法无需匹配，保留全部样本：
- **IPW（逆概率加权）**：通过加权而非丢弃样本来平衡协变量分布
- **DR（双重稳健估计）**：结合结果回归和 IPW，只需其中一个模型正确即可一致估计
- **OR / DML**：进一步使用机器学习提升模型灵活性
