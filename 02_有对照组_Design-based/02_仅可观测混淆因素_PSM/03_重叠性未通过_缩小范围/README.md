# 重叠性未通过 → 缩小结论适用范围

## 状态

PSM 的**重叠性检查未通过**：处理组与对照组的倾向得分分布重叠不足，直接估计 ATE 会依赖外推，不可信。

## 决策分支

```
重叠性未通过
  ├── 是否可缩小结论适用范围以避免外推？
  │     ├── Y → 重新定义待估目标参数
  │     │       ├── 是否有明确的业务分组？
  │     │       │     ├── Y → 仅保留重叠性良好的分组匹配估计（见子目录 分组匹配估计/）
  │     │       │     └── N → 转向估计局部参数 LATE（见子目录 局部参数LATE/）
  │     └── N → 直接使用 IPW / DR → OR / DML（见上级 02_IPW_DR/）
```

## 核心思想

与其强行估计整体 ATE（Average Treatment Effect），不如：
- **缩小目标人群**：只在有充分重叠的子群体上估计 ATT
- **转换参数目标**：从 ATE/ATT 转向 LATE（Local Average Treatment Effect），仅估计"受干预边际影响的人群"的效应

## 子目录说明

| 子目录 | 方法 | 说明 |
|--------|------|------|
| `分组匹配估计/` | 分组 PSM | 识别重叠性良好的业务分组，仅在该分组估计效应 |
| `局部参数LATE/` | LATE / RDD / IV | 利用自然断点或工具变量估计局部因果效应 |
