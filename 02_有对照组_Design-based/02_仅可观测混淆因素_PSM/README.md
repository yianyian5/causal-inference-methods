# PSM 路径 — 仅存在可观测混淆因素

## 适用条件

- 有对照组
- **不存在明显的不可观测混淆因素**（或可以合理假设可忽略性）
- 关键识别假设：**条件可忽略性（CIA / Unconfoundedness）**：在给定可观测协变量 $X$ 的条件下，处理分配与潜在结果独立

## 决策分支

```
仅可观测混淆因素 → PSM
  ├── 通过重叠性检查（Overlap Check）？
  │     ├── Y → 是否可接受样本损失？
  │     │       ├── Y → 通过平衡性检查（Balance Check）？
  │     │       │       ├── Y → 进入稳健性检验 ✅
  │     │       │       └── N → 改用 IPW / DR（见子目录 02）
  │     │       └── N → 改用 IPW / DR（见子目录 02）
  │     └── N → 是否可缩小结论适用范围？
  │               ├── Y → 重新定义目标参数（见子目录 03）
  │               └── N → 直接使用 IPW / DR → OR/DML（见子目录 02）
```

## 子目录说明

| 目录 | 方法 | 适用场景 |
|------|------|----------|
| `01_PSM匹配/` | Propensity Score Matching | 重叠性好，可接受样本损失 |
| `02_IPW_DR/` | IPW、DR、OR、DML | 不愿损失样本，或平衡性不达标 |
| `03_重叠性未通过_缩小范围/` | 分组匹配、LATE、RDD、IV | 重叠性差，通过缩小范围或局部参数处理 |
