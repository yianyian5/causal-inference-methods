# 有对照组 — Design-based 方法

## 适用条件

**可以找到对照组**，利用设计上的随机性或准随机性识别因果效应。

## 决策分支

```
有对照组
  ├── 是否存在明显不可观测的混淆因素？
  │     ├── Y → DID 路径（见子目录 01）
  │     └── N → PSM 路径（见子目录 02）
```

## 子目录说明

| 目录 | 方法族 | 关键识别假设 |
|------|--------|------------|
| `01_不可观测混淆因素_DID` | DID / SCM / SDID | 平行趋势假设（Parallel Trends） |
| `02_仅可观测混淆因素_PSM` | PSM / IPW / DR / DML / RDD / IV | 条件可忽略性（CIA）或局部随机性 |
