# 稳健性检验

## 位置说明

所有因果推断方法在得出主要估计结果后，都需要汇聚到此阶段进行**稳健性检验**，才能最终输出可信结论。

## 稳健性检验的四个维度

### 1. 基于经济理论与业务直觉的合理性验证

- 估计结果的**方向和量级**是否符合业务预期？
- 是否存在可以解释结果的经济学/业务逻辑？
- 效应异质性（分人群、分渠道）是否与直觉一致？

### 2. 数据描述性验证

- 处理组与对照组的**基本特征**是否符合预期？
- 结果变量在处理前后的**趋势图**是否合理？
- 是否存在明显的数据质量问题（异常值、缺失值模式）？

### 3. 安慰剂检验（Placebo Test）

| 类型 | 做法 | 目的 |
|------|------|------|
| **时间安慰剂** | 将"处理时间"提前到处理前某期，看是否仍有显著效应 | 排除时间趋势干扰 |
| **人群安慰剂** | 对理论上不应受影响的子群体（对照组内部）重复分析 | 排除系统性偏差 |
| **结果变量安慰剂** | 对理论上不应被影响的指标做同样分析 | 排除整体宏观趋势 |
| **随机置换检验** | 随机打乱处理标签，重复估计，构造参照分布 | 评估统计推断的可靠性 |

### 4. 敏感性分析（Sensitivity Analysis）

- **参数规格变化**：更换控制变量集、函数形式、样本范围
- **匹配方法变化**（PSM 路径）：更换匹配算法、带宽、卡尺
- **带宽敏感性**（RDD 路径）：在不同带宽下结果是否稳健
- **Rosenbaum Bounds**（PSM 路径）：对不可观测混淆因素的敏感性上界
- **假设违反情景**：若关键假设略有偏离，结论是否仍成立

## 通过标准

稳健性检验**没有统一标准**，需综合判断：
- 安慰剂检验不显著（或效应极小）✅
- 主要结论在不同规格下方向一致，量级相近 ✅
- 结果符合业务理论逻辑 ✅
- 对合理范围内的假设偏离不敏感 ✅
