# 因果推断方法选用框架

本目录根据 `framework_flowchart.md` 中的方法选用流程图，按**决策逻辑**组织各因果推断方法，每个叶子节点（具体方法）对应一个子目录。

---

## 目录结构总览

```
causal-inference-methods/
│
├── framework_flowchart.md          # 原始流程图（Mermaid）
├── framework_flowchart.py          # 流程图生成脚本
│
├── 01_无对照组_Model-based/         # 无法找到对照组 → 模型驱动
│   ├── 01_经济学模型_结构估计/       # 需要业务洞察 → BLP、结构估计等
│   └── 02_统计学模型/               # 不需要业务洞察 → BSTS / Causal Impact
│
├── 02_有对照组_Design-based/        # 可以找到对照组 → 设计驱动
│   ├── 01_不可观测混淆因素_DID/      # 存在不可观测混淆 → DID 家族
│   │   ├── 01_经典DID/
│   │   │   ├── TWFE/                # 双向固定效应估计
│   │   │   │   ├── 通过事前趋势检查/         ✅ 结果可信
│   │   │   │   ├── 未通过事前趋势_调整后重试/ ⚠️ 调整平行趋势
│   │   │   │   └── 无法调平行趋势_转SCM_SDID/ ❌ 转合成控制
│   │   │   └── SCM_SDID/            # 合成控制 / 合成双重差分
│   │   └── 02_非经典DID_Staggered/  # 交错处理时间的 DID
│   │
│   └── 02_仅可观测混淆因素_PSM/      # 仅可观测混淆 → PSM 家族
│       ├── 01_PSM匹配/              # 倾向得分匹配
│       │   ├── 通过重叠性和平衡性检查/       ✅ 结果可信
│       │   └── 重叠性或平衡性未通过_转IPW_DR/ ❌ 转加权方法
│       ├── 02_IPW_DR/               # 逆概率加权 / 双重稳健
│       │   └── OR_DML/              # 结果回归 / 双重机器学习
│       └── 03_重叠性未通过_缩小范围/  # 重叠性差 → 缩小适用范围
│           ├── 分组匹配估计/          # 有业务分组 → 分组 PSM
│           └── 局部参数LATE/         # 无业务分组 → 局部效应
│               ├── RDD/             # 有自然断点且无操纵 → 断点回归
│               ├── Bunching分析/     # 有断点但存在操纵 → 密度堆积
│               └── IV工具变量/       # 无断点但有工具变量 → 2SLS
│
└── 03_稳健性检验/                   # 所有方法的最终汇聚点
```

## 决策流程速查

```
全量评估开始
    │
    ├─ 能找到对照组？
    │   │
    │   ├─ N → Model-based（见 01/）
    │   │       ├─ 需要业务洞察？ Y → 经济学模型
    │   │       └─ 需要业务洞察？ N → 统计学模型 (BSTS)
    │   │
    │   └─ Y → Design-based（见 02/）
    │           │
    │           ├─ 存在不可观测混淆？
    │           │   ├─ Y → DID
    │           │   │       ├─ 经典DID？ Y → TWFE → Event Study
    │           │   │       │                   通过 → 稳健性检验
    │           │   │       │                   未通过 → 调整 or SCM/SDID
    │           │   │       └─ 经典DID？ N → Staggered DID
    │           │   │
    │           │   └─ N → PSM
    │           │           ├─ 重叠性通过？ Y → 样本损失可接受？
    │           │           │                   Y → 平衡性通过？ Y → 稳健性检验
    │           │           │                                    N → IPW/DR → OR/DML
    │           │           │                   N → IPW/DR → OR/DML
    │           │           └─ 重叠性通过？ N → 可缩小范围？
    │           │                               Y → 有业务分组？
    │           │                               │     Y → 分组匹配
    │           │                               │     N → LATE
    │           │                               │           ├─ 有自然断点？
    │           │                               │           │   Y → 操纵？ N→RDD  Y→Bunching
    │           │                               │           │   N → 有IV？ Y→IV  N→分组匹配
    │           │                               └─ 不可缩小 → IPW/DR → OR/DML
    │
    └─ 所有方法 → 稳健性检验（见 03/）→ 结束
```

## 各方法估计目标对比

| 方法 | 估计目标 | 识别假设 |
|------|----------|----------|
| BSTS / Causal Impact | ATE（时间序列） | 处理前拟合的反事实 |
| BLP / 结构估计 | 结构参数（弹性等） | 经济学理论模型 |
| TWFE / DID | ATT | 平行趋势 |
| SCM / SDID | ATT（合成控制） | 近似平行趋势 |
| Staggered DID | Group-time ATT | 平行趋势（分队列） |
| PSM | ATT | 条件可忽略性 + 重叠性 |
| IPW / DR | ATE 或 ATT | 条件可忽略性 |
| DML | ATE / CATE | 条件可忽略性（高维） |
| RDD | LATE（断点处） | 局部随机（无操纵） |
| Bunching | 行为弹性 | 密度反事实 |
| IV / 2SLS | LATE（Compliers） | 工具变量三条件 |
